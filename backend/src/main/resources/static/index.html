API Endpoints
GET /api/dashboard Weekly metrics, category breakdown, recent transactions
POST /api/receipts/upload Upload receipt image → parsed by Spring AI (GPT-4o-mini)
GET /api/receipts/{id} Single receipt with all items
GET /api/expenses All receipts, sorted by date descending
GET /api/budget Current weekly and category budgets
PUT /api/budget Update weekly budget {"weeklyAmount": 250.00}
PUT /api/budget/categories Update per-category limits {"Produce": 70.00, ...}
GET /api/analytics Chart data + stats — ?period=week|month|3months
